## FILE GUIDE

이 문서는 저장소에 포함된 주요 파일의 역할을 간단히 정리한 안내 문서입니다.

#### 1. `preprocess.py`

얼굴 이미지 전처리 스크립트입니다.

##### 주요 역할

- YOLOv8-face 모델 로드
- 원본 이미지에서 얼굴 검출
- 검출된 얼굴 영역 crop
- 224x224 크기로 리사이즈
- 전처리 결과 저장

##### 사용 목적

분류 모델 입력 품질을 맞추기 위한 전처리 단계입니다.

---

#### 2. `classfication.py`

데이터 분할 스크립트입니다.

##### 주요 역할

- 감정 클래스별 이미지 폴더 순회
- 이미지 셔플
- train / valid / test 분할
- 분할 결과를 폴더별로 저장

##### 현재 기준

- train: 0.8
- valid: 0.1
- test: 0.1

##### 참고

파일명에 오타가 있으나, 원본 실험 파일명을 유지하고 있습니다.

---

#### 3. `cross-validation.py`

감정 분류 모델 학습 스크립트입니다.

##### 주요 역할

- 5-fold cross-validation 설정
- ResNet-18 로드
- 감정 7개 클래스 분류 레이어 구성
- fold별 학습 및 검증
- 모델 가중치 저장

##### 사용 목적

감정 분류 성능 실험을 수행하기 위한 학습 단계입니다.

##### 참고

일부 객체(`image_datasets`, `train_model`)는 별도 정의가 필요합니다.

---
